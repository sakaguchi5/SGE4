# Spiral 7 CU4 mutation matrix

The first invocation applies five independently rehashed mutations to each A/B/C Raw Candidate:

1. proposed Artifact identity
2. write-set authority identity
3. Dispatch group count
4. legal write-set class
5. representation-resource contract identity

A sixteenth attack corrupts the stored Raw Candidate identity without rehashing. All must be rejected by the independent Family Verifier. A Verified Candidate from invocation zero must also be rejected when replayed against invocation one.
