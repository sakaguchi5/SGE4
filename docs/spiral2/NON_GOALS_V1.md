# Spiral 2 non-goals v1

Spiral 2 does not implement skinning, vertex influence, clips, blending, IK,
constraints, scale, temporal history, motion vectors, variable bone count or
topology, DispatchIndirect, variable-length buffers, runtime variant selection,
conditional regions, Texture Flow, streaming, residency, multiple devices or
adapters, partial recovery, general PGA multivectors, arbitrary shader proofs,
or the capability after Spiral 2.

No speculative type, reserved operation, capability bit, or empty extension
hook may be introduced for those subjects.
