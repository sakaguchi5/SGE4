## Cross-Spiral common structure

Every successful Spiral converged on the same pattern:

```text
Meaning
  -> Candidate alternatives
  -> independent verification
  -> Frozen identity
  -> equivalent observation
  -> Recovery-preserving runtime
  -> optional Adapter-specific measurement
```

This common structure, rather than any one Candidate implementation, is the principal input to Level 4 v2.
