# Spiral 1 Architecture Final Freeze V1

## Completion boundary

After CU5 passes, the following chain is the frozen Spiral 1 architecture:

```text
Canonical PGA Semantic
  -> Raw Matrix / Direct PGA Candidates
  -> Independent Representation Verifier seals
  -> verifier-bound Frozen Representation Leaves
  -> existing verified ExecutionPlan authority
  -> Four-Leaf Level 4 v1 Frozen Composition
  -> Frozen-only Runtime
  -> Observation Contract V2
```

## Qualified invariants

CU5 closes S1-I01 through S1-I16. S1-I17 (fair real-GPU measurement) and S1-I18 (evidence-based next Level 4 extension decision) remain owned by CU6.

## Recovery unit

The recovery unit is the whole Frozen Composition. No Leaf instance, shared Buffer, completion token or device-epoch object survives reconstruction. Verified constant bytes remain part of the higher Frozen Comparison artifact and are rebound without Runtime selection.

## Actual removal rule

The removed domain remains quarantined in `AwaitingAdapter` while its LUID is excluded. If WARP is the only eligible adapter, same-process reacquisition must remain unavailable; this is a valid and required quarantine result, not a recovery failure. After the removal process exits, a separate fresh process rematerializes the same Frozen artifact and must reproduce the same Observation V2 result. The old domain may not submit or validate stale handles.

## Non-changes

CU5 does not change Schema 17, Runtime 17, the Level 4 v1 Buffer-only static DAG, the PGA Motor convention, either lowering, the S00-S15 corpus, or Observation Contract V2.
